# Journey to Nana

**Journey to Nana** is a narrative, slice-of-life pixel art game developed using Godot Engine 4.x. 

> **A Personal Project:** This game is a special, personal passion project created by Indra dedicated to his girlfriend. It follows the daily college life of Indra as he navigates his routines, friendships, and personal stories with the people around him.

## Project Status

🚧 **In Development (Playable, Active Prototype)** 🚧

The game is currently in active development. Several days (Act 1) are already playable. Core mechanics such as the quest system, scene transitions, save/load features, and main interactions have been implemented. Content creation, balancing, and UI polishing are ongoing.

## Gameplay Summary

- **Exploration:** Wander around daily environments (dorm room, campus, etc.).
- **Interaction:** Interact with everyday objects (doors, buckets, cupboards, phones, PC).
- **Daily Quests:** Complete daily routines like taking a bath, changing clothes, and going to class.
- **Narrative Dialogue:** Rich storytelling powered by the Dialogic plugin.
- **Timeskips:** Auto and manual timeskips to progress through the day.
- **Meaningful Choices:** Dialogue options and interactions that influence mood and story elements.

## Design Pillars

- **Personal & Relatable Narrative:** Capturing the essence of college life, daily struggles, and relationships.
- **Simple Interactions:** Emphasizing a relaxing, slice-of-life vibe over complex gameplay mechanics.
- **Realistic Scope:** A short but dense experience with high replayability through different choices.

## MVP Scope

- **Act 1 (Surabaya):** Covers Day 1 & Day 2.
- Fully integrated daily quest and timeskip systems.
- Main object interactions (doors, phones, PC, bed, etc.).
- Robust Dialogic integration for narration and conversations.
- Simple endings for each day.
- Fully working Save/Load and dynamic scene transition systems.

## Tech Stack

- **Engine:** Godot 4.x
- **Genre:** 2D Pixel Art, Narrative, Slice of Life
- **Platform:** PC (Windows)
- **Notable Implementations:** Custom Shader materials (Iris Reveal transitions), advanced Custom Resource data handling, and Finite State Machines (FSM).
